## Русские шашки 10×8 (pygame) + RL против минимакса

Проект: шашки на доске **10 столбцов × 8 строк** (игровые клетки — тёмные), визуал на **pygame**.

Бот обучается **Q-learning** против **минимакса** (глубина настраивается), обучение сохраняется в файл и загружается при старте, чтобы не тренироваться каждый раз.

### Установка

```bash
python -m pip install -r requirements.txt
```

### Запуск

```bash
python main.py
```

В меню можно:
- играть против RL-бота или минимакса
- открыть **отдельное окно обучения** (`T` или `3`): настройки, прогресс-бар, остановка, автосохранение в `qtable.pkl`
- запустить **ДЕМО**: RL (белые) против минимакса (чёрные) — пункт `4`
